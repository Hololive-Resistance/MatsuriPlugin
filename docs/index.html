Calculator

The calculator feature compiles data from the CoreProtect database to provide a much simpler way to track Holomatsuri progress using it's own points system. It provides commands to show player points, leaderboards and other supporting output features which would otherwise be tedious to do by hand.

The calculator may require its own database to be determined by whether querying the CO database frequently is safe.
